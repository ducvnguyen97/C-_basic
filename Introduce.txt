1.What is C++
    C++ is cross platform language that can be used to create high-perfomance applications
    C++ was developed by Bjarne Stroustrup.
    C++ give programmers a high level of control over system resource and memory
2.Why use C++
    C++ can be found in operate systems, Graphical user interfaces and embedded systems
    C++ is an object-oriented programming language which give a clear structure to programe and allow code to be reused, lowering developments cost.
    C++ is close C# and Java.
3.Difference between C and C++
    C++ was developed as an extension of C, and both languages have almost the same syntax.
    The main difference between C and C++ support class and object, while C does not.
